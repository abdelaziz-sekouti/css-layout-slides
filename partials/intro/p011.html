Gérer les débordement sur l'axe principal

Par défaut, les éléments sont alignés le long de l'axe principal. Les éléments peuvent dépasser la largeur du conteneur.

Pour permettre aux éléments de s'afficher sur plusieurs lignes, on utilise l'attribut flex-wrap. Sa valeur par défaut est nowrap

  • flex-wrap: wrap; /* Permet le renvoie à la ligne. */

Gérer l'ordre des éléments

On utilise pour l'élément l'attribut order. Celui-ci vaut par défaut 0 pour tous les éléments

Si on applique au premier élément de la liste order: 1;, celui-ci apparaîttra en fin de liste.
