Gérer l'alignement sur l'axe secondaire (Cross Axis)

L'alignement sur l'axe secondaire est géré en appliquant au conteneur l'attribut :
align-items: stretch; (Etendre) Valeur par défaut.

Cet attribut peut prendre les valeurs suivantes :

  • stretch /* Etendu le long de l'axe secondaire */
  • flex-start /* Début de l'espace */
  • center /* ! */
  • flex-end /* Fin de l'espace */
  • baseline /* Se base sur le texte pour l'alignement */

On peut également changer ce comportement à un élément particulier en lui appliquant l'attribut align-self avec l'une des valeurs précédentes.
