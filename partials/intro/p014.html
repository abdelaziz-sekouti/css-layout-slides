Gérer la distribution sur l'axe principal (Main Axis)

Trois propriétés sont utilisées au niveau des éléments :

  • flex-grow: <Nombre> ; /* définit comment l'élément va s'étendre */
  • flex-shrink: <Nombre> ; /* définit comment l'élément va se rétrécir */
  • flex-basis: <taille> ; /* définit la taille occupée par l'élément */

Ces trois attributs peuvent être donnés en un seul coup avec l'attribut flex :

  • flex: 2 1 auto;

Dans l'exemple précédent, la valeur de flex-grow est 2. Cet élément prendra deux (2) parts de l'espace libre pour grandir.

flex-basis est défini à auto, le moteur de rendu se basera sur la taille du contenu pour faire ses calculs.
